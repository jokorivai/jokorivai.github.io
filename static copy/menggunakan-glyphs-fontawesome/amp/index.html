Menggunakan Glyphs/Ikon FontAwesome

by Joko Rivai

2019-03-13

FontAwesome hanya contoh, Anda dapat juga menggunakan font lain seperti Ionicon dan Material Design Icons.

Konsep

Untuk dapat melihat penggunaan ikon pada aplikasi yang dibangun dengan TabrisJS, kita perlu memikirkan cara paling mudah dalam penerapan font yang berisi icon ke dalam aplikasi. Untuk ini ide saya adalah menggunakan listview yang tiap itemnya menampilkan nama ikon dalam FontAwesome serta tampilan ikonnya.

Perlu diperhatikan bahwa aplikasi TabrisJS berbeda dengan aplikasi Web atau aplikasi hibrid yang menggunakan HTML yang di-compile dan ditampilkan dalam WebView pada aplikasi Cordova lainnya. Pada TabrisJS kita tidak dapat menggunakan HTML, karenanya kita memerlukan cara lain untuk menampilkan icon dalam FontAwesome pada aplikasi kita. Caranya, misalnya, dengan mengijinkan developer menampilkan teks yang di mix/dicampur dengan nama icon atau class icon kemudian utilitas yang kita siapkan akan memeriksa teks tersebut dan mengganti/mereplace nama-nama class dalam teks dengan icon sesuai kode karakternya.

Kita perlu menyiapkan sebuah file javascript sebagai utilitas untuk mem-parse teks dan menggantinya dengan icon pada file terpisah, sehingga dapat digunakan kembali pada proyek aplikasi mobile yang dibangun dengan TabrisJS di masa mendatang.

Membuat Projek

Langkah pertama, silahkan buat sebuah proyek dengan mengikuti petunjuk di sini.

Kemudian dalam folder proyek buatlah sebuah folder dengan nama res, Dan didalam folder res ini buatlah sebuah folder lagi dengan nama font. Download & copy file FontAwesome (TTF) ke dalam folder res/font ini, ganti namanya agar mudah dibaca. Perhatikan struktur folder berikut:

File di atas berisi objek (instance dari sebuah fungsi). Objek ini melakukan pemanggilan fungsi

tabris.app.registerFont('FontAwesome', './res/font/fontawesome-webfont.ttf#'+this.font);

sehingga Anda harus memastikan bahwa nama file dan struktur folder benar-benar sesuai dengan yang ada pada kode di atas.

Kode selanjutnya adalah fungsi fa( ) yang menerima parameter string, yang kemudian memeriksanya dan mengganti class-class ikon FontAwesome dengan karakter ikon yang sesuai. Perhatikan bahwa kode ret = ret.split('{'+glyph.n+'}').join(glyph.c); memerlukan kurung kurawal untuk menandai class ikon, contoh:

  • fa-map harus ditulis {fa-map}
  • fa-cog harus ditulis {fa-cog}

Maksud konvensi penulisan ini agar class yang direplace menjadi karakter icon hanya class yang benar-benar akan direplace, dan menghindari proses replace pada string yang kebetulan berisi teks yang sama dengan nama class. Dengan konvensi ini, developer dapat membuat string seperti berikut:
let Teks = 'Class fa-hourglass akan menghasilkan icon {fa-hourglass}';

Selanjutnya untuk diperhatikan, setiap widget/komponen UI yang menggunakan utilitas ini harus menggunakan font FontAwesome untuk property Text-nya:

new TextView({
  right: 10, centerY: 0, width: 140,
  font: 'bold 12px FontAwesome',
  textColor: '#212121'
  text: TabrisFontAwesome.fa(
    'Class fa-hourglass akan menghasilkan icon {fa-hourglass}'
  );
});

Kode Utama

Setelah kita siapkan utilitas, mari kita buka file app.js dan ganti isinya dengan kode berikut:

const {
  NavigationView, 
  Page, 
  CollectionView, 
  Composite, 
  TextView, 
  ui
} = require('tabris');

// Include utilitas:
const TabrisFontAwesome = require('./TabrisFontAwesome');

let navigationView = new NavigationView({
  left: 0, top: 0, right: 0, bottom: 0
}).appendTo(ui.contentView);

let page = new Page({title: 'FontAwesome Icons'}).appendTo(navigationView);
let glyphs = TabrisFontAwesome.glyphTable;
new CollectionView({
  left:0, top:0, bottom:0, right:0,
  cellHeight: 48,
  itemCount:glyphs.length,
  createCell: function(){
    let cell = new Composite();
    let f = '14px '+TabrisFontAwesome.font;
    new Composite({
      id: 'garis',
      left: 0, right: 0, bottom: 0, height: 1,
      background: '#e7e7e7'
    }).appendTo(cell);
    new TextView({
      id: 'glyph',
      alignment: 'right',
      right: 10, centerY: 0, width: 140,
      font: f,
      textColor: '#212121'
    }).appendTo(cell);
    new TextView({
      id: 'name',
      left: 10, right: ['prev()', 10], centerY: 0,
      font: f,
      textColor: '#212121'
    }).appendTo(cell);
    return cell;
  },
  updateCell: function(cell, index){
    let glyph = glyphs[index];  
    cell.apply({
      '#name': {text: glyph.n},
      '#glyph': {text: TabrisFontAwesome.fa('{'+glyph.n+'}')}
    });   
  },
}).appendTo(page);
